## Linux 的 curl 命令 
### 介绍

基本使用：

``` bash
curl www.baidu.com
```

#### 设置 headers

``` bash
curl www.baidu.com -H 'Content-Type: application/json'
```

curl 是个多功能命令行工具，它的常见功能有：

* 下载

### 下载文件

``` bash
curl -O "https://curl.se/logo/curl-logo.svg"
```
